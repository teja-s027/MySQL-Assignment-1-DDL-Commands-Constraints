MySQL Assignment 1 – DDL Commands & Constraints
📌 Project Overview
This project demonstrates the use of MySQL Data Definition Language (DDL) commands and SQL constraints. It includes database creation, table creation, table modification, renaming, truncating, dropping, and applying constraints.
